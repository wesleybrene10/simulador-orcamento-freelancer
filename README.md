Simulador de Orçamento Freelancer
Sobre o projeto

Este projeto é um simulador de orçamento para serviços de freelancer. O usuário pode selecionar diferentes serviços e visualizar o valor total automaticamente.

Funcionalidades
Seleção de serviços
Cálculo automático do valor total
Exibição dos serviços selecionados
Botão para limpar seleção
Tecnologias utilizadas
HTML
CSS
JavaScript
Como usar
Abra o arquivo index.html no navegador
Selecione os serviços desejados
O valor total será atualizado automaticamente
